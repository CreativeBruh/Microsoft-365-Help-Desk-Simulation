## 🧩 Project Overview

This lab walks through the complete setup of a Microsoft 365 tenant designed to simulate a functioning organization.  
The workflow includes:

### 1. Tenant Initialization
Setting up a clean Microsoft 365 environment and confirming administrative access.

### 2. User Lifecycle Creation
Building out employee accounts, assigning licenses, and validating first‑time sign‑ins.

### 3. Organizational Structure
Defining job titles, departments, and manager relationships to create a realistic directory.

### 4. Shared Mailboxes
Configuring department mailboxes with delegated access for team‑based communication.

### 5. Mail Flow Integration
Redirecting support emails to the Spiceworks Cloud Help Desk using an Exchange transport rule while preserving the original sender.

### 6. Ticket Creation Validation
Confirming end‑to‑end functionality through message trace and automatic ticket generation.
