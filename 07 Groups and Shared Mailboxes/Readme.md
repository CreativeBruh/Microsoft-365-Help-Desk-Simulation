07 – Groups and Shared Mailboxes

This step documents the creation and configuration of shared mailboxes within the Microsoft 365 Admin Center. Shared mailboxes are essential for department‑wide communication, delegated access, and help desk workflows. They allow multiple users to read and respond to messages from a common address without requiring additional licenses.
